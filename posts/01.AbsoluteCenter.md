
## 01.Absolute Center
<br>

요소를 가운데 위치 시킵니다.
<br><br>

```css
div {
    position: relative;
    height: 300px;
    border: 1px solid pink;
}
h1 {
    position:absolute;
    top: 50%;
    left: 50%;
    margin: 0;
    transform: translate(-50%,-50%);
}
```
