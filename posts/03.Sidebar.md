
## 03.Sidebar

***grid-template-column: minmax(min, max)*** <br>
최소 크기와 최대 크기를 가진 사이드바를 만듭니다.

```css
.parent {
    display: grid;
    grid-template-columns: minmax(150px, 25%) 1fr;
}	
```
